# Chapter 2: Operators & Expressions 🚀

### 1. Arithmetic Operators
- `+`, `-`, `*`, `/`
- `%` (Modulo): Returns the remainder (e.g., `10 % 3 = 1`).

### 2. Assignment Operators
- `=` (Assign)
- `+=`, `-=`, `*=`, `/=` (e.g., `a += 5` is `a = a + 5`).

### 3. Comparison Operators
- `==`, `!=`, `>`, `<`, `>=`, `<=` (Always returns true/false).

### 4. Logical Operators
- `&&` (AND), `||` (OR), `!` (NOT).
